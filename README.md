# obtenerUFSIIPy
Obtener uf del dia, por a単o, mes y dia... si no se envia parametros se obtiene la uf del dia actual.

Script desarrollado con Python3.

Paquetes instalar los paquetes utilizados, use la siguiente orden

pip install requests beautifulsoup4 lxml

para ejecutarlo use la siquiente orden

Si necesitas obtener la UF de un dia en especifico del presente mes y a単o, puedes hacerlo de la siguiente manera:

```
python3 script.py 30
```
Si necesitas obtener la UF de un dia y un mes del presente a単o, puedes hacerlo de la siguiente manera:

``` 
python3 script.py 30 12
```
Si necesitas obtener la UF de un dia, mes y a単o en especifico, puedes hacerlo de la siguiente manera:

```
python3 script.py 30 12 2020
```

En caso de no enviar parametros, se obtiene la UF del dia actual.
